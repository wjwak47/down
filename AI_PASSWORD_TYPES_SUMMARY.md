# PassGPT 生成的密码类型 - 快速回答

## 简单回答

PassGPT 生成的密码是 **真实用户最常用的密码**，因为它在 **3200万真实泄露密码**（RockYou 数据集）上训练。

## 最常见的 7 种密码类型

### 1️⃣ 名字 + 数字 (25-30%)
```
joe377
lopez85
mike1990
sarah2000
john123
```
→ 英文名字 + 年份/幸运数字

### 2️⃣ 常见单词 + 数字 (20-25%)
```
password123
love2009
sunshine88
monkey123
dragon99
```
→ 高频单词 + 简单数字

### 3️⃣ 日期相关 (10-15%)
```
jan3030
july4joel
dec2525
may1990
```
→ 月份 + 日期 + 年份

### 4️⃣ 运动/爱好 (8-10%)
```
soccer12
football1
baseball7
guitar123
```
→ 运动/爱好 + 号码

### 5️⃣ 键盘模式 (5-8%)
```
qwerty123
asdfgh
123456789
qazwsx
```
→ 键盘连续按键

### 6️⃣ 流行文化 (5-8%)
```
pokemon1
starwars
superman
batman123
```
→ 游戏/电影/漫画

### 7️⃣ 电话号码 (3-5%)
```
5551234
8675309
```
→ 7-10位数字

## 密码特征

| 特征 | 说明 |
|------|------|
| **长度** | 6-10个字符（最常见8个） |
| **字符类型** | 50%是小写+数字，30%纯小写，15%大小写混合 |
| **语义** | 有意义的单词（不是随机字符串） |
| **记忆性** | 符合人类记忆习惯 |
